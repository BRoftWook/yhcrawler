# Crawler for Yonhap News

## Requirements

- Java (JDK 1.6 +)

## Run

- Go Run Configurations > Arguments > Program Arguments
- Type article published date and save directory you want as below

		-f [Directory] -d [Date]

  for example,  `-f C:\Users\Wook\Desktop -d 2015-01-04`

- Run the code
